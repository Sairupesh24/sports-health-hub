# ISHPO - Integrated Sports Health & Physio Operating System

## Project Overview

ISHPO is a comprehensive operating system designed for sports health professionals to streamline clinical workflows, track patient progress, and manage practices efficiently. It is a product of **CSSH (Center For Spine and Sports Health)**.

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm

### Installation

1. Clone the repository:
   ```sh
   git clone <repository-url>
   ```

2. Install dependencies:
   ```sh
   npm install
   ```

3. Start the development server:
   ```sh
   npm run dev
   ```

## Technologies Used

- **Frontend**: React, TypeScript, Vite
- **Styling**: Tailwind CSS, shadcn/ui
- **Backend/Database**: Supabase
- **Icons**: Lucide React
- **Charts**: Recharts

## Deployment

To deploy the project, you can use platforms like Vercel, Netlify, or any other static site hosting service. Ensure you configure the necessary environment variables for Supabase.
